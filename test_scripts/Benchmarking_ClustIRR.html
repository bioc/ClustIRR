ClustIRR Benchmarking

TL;DR

Confirmed

  • Motifs successfully detected accross all versions
  • Global similarites also found (same algorithm for all versions)
  • OvE values for found motifs of v1/2 roughly comparable to ground truth

To be investigated

  • p-value of v1 seems to be lacking accuracy compared to v2/3
    • leads to fdr of version 1 being lower than v2/3
    • leads to v1 detecting more motifs than v2/3 (effectively lowered detection threshold)
  • OvE values for found motifs of v3 partially different than ground truth
    • high value in v3 (i.e.Â 275.762251) against infinity in gt
    • maybe caused by FET implementation vs formula used for GT
